# SHTI (ПРХТ)

S - simple 
H - hash
T - table
I - implementation

# Описание 
	Высокопроизводительная реализация хеш таблиц на языке программирования C++.
	Не использует стандартные контенеры STD. Была разработана в качетсве копмактной библеотеки для внедрения 
	в прочие проекты.

fix - исправления ошибок

test - тесты

doc - обновление документации
add - нововведения
